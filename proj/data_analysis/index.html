Project 3: Data Analysis

Introduction

In this project you will curate and process actual public data to answer questions of your choice. The goal of the project is to apply data science to answer actual questions using data and the techniques you've learned in CS88 and Data 8.

Logistics

This is a 14-day project. You may work with one other partner. You should not share your code with students who are not your partner or copy from anyone else's solutions.

In the end, you will submit one project for both partners. The project is worth 20 points.

You will turn in the following files:

  • data_analysis.html

You can submit by uploading these files to OK.

You will be able to view your submissions on the OK dashboard.

Logistics

This is a 14-day project. You may work with one other partner. You should not share your code with students who are not your partner or copy from anyone else's solutions. In the end, you will submit one project for both partners. The project is worth 20 points.

Task 1: Obtain the Data

Go to http://catalog.data.gov/dataset and select one or more dataset(s) of your choice in the format(s) of your choice. We suggest working with data that is available in .csv or excel formats as these can be easily processed. You are allowed to work with other data types as well, as long as you know how to read and process the data. Please save a link to your dataset and submit that as part of your Jupyter notebook.

You may also work with multiple datasets in order to explore relationships between the two datasets, or to just have a greater amount of data to work with. We suggest that you use no more than a few datasets. Rather than adding complexity through the volume of data, we suggest adding complexity through your analysis of the data.

Task 2: Pose Three Questions

Ask three questions of varying complexity (easy, medium, difficult) that you want to answer with the dataset. An easy question is something that can be answered by looking at one column of the data (i.e. what is the mean value of some column?). The medium questions should require complex processing of the data in order to arrive at an answer. The hard questions should also require complex processing. With more difficult questions, it's okay if you're unable to arrive at concrete answers. The emphasis is on the process used to process the data and arrive at some sort of conclusion. These guidelines leave a lot of room to exercise creativity and interest. We encourage you to come up with ideas by the final lab meeting (11/28/16) and discuss your ideas with a TA. As you are processing the data (next task), you may find that you want to change your questions, which is fine. These questions should be written up in the text portion of the Jupyter notebook.

Task 3: Read in the Data

Process the dataset through methods of your choice. You should start by reading the data into some sort of data structure. One great choice would be to use the Tables module used in Data 8. You can take a look at readings and assignments for Data 8 to see examples of data processing. You are also welcome to use external Python libraries to analyze the data. Feel free to look up methods of reading and processing data files. Some helpful links are listed below.

  • Python CSV tools
  • Pandas CSV tools

Task 4: Analyze the Data

We expect to see some use of techniques you have learned in Data 8 and CS88, listed below. However, you are not limited to these techniques. You can use other techniques that were not taught in cs interpret the data. All techniques used be part of the Jupyter notebook you submit. Code and appropriate captions/explanations should be part of the notebook.

  • Map/reduce/filter
  • Iterative/recursive algorithms
  • Higher order functions
  • Table joins
  • Random sampling
  • Linear regression
  • Estimating confidence intervals
  • Hypothesis testing
  • Classification
  • Plots/historgrams
  • Any other techniques learned in Data8/CS88

Task 5: Answer the Questions

Answer the questions you posed in task 2 as best as you can with the data and your current knowledge of data science. As mentioned above, you are not expected to provide concrete answers to all questions. However, you are expected to provide a greater level of understanding and insight to the questions through data analysis. Please write your answers out in the text portion of the Jupyter notebook.

Grading

  • Asking three questions about the data (3 pts)
  • Complexity of the questions (2 pts)
  • Data analysis (code and text) (9 pts)
  • Answering the questions (text) (6 pts)

To get a good idea of what we expect the final Jupyter notebook to look like, please take a look at the Data 8 lab assignments. The labs have both code and text explanations that accompany the code. You do not need to describe every single line of code, but be sure to explain the techniques you are using to analyze the data.
